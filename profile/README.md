## AreaPulse - 데이터베이스응용 3조

> 가격 중심의 기존 서비스 대신 <br>
지도 기반으로 소음, 치안, ​교통 혼잡도, 인프라와 같은 실제 거주 만족도에 영향을 주는 정보를 한눈에 
볼 수 있는 앱 개발​ 


<br>

## 👥 멤버

| 멤버 | 박현규 | 문형일 | 배용빈 | 뚜먁튜 |
|:------:|:------:|:------:|:------:|:------:|
| 기여 | iOS 앱 개발 <br> API 명세서 작성 <br> API 오류 해결 | 환경 구성(DevOps) <br> 3기가 DB 데이터 수집 및 정제(DBA) <br> API 오류 해결(백엔드개발)  | API 초안 작성 | API 초안 작성<br> 데이터 조사 |
| 퍼센트 | 35% | 35% | 20% | 10% |

## 📱 소개
**지도 기반 탐색**
  <br>원하는 좌표에서 반경 내의 건물, 인프라 검색​<br><br>
**건물 정보 상세 조회**
  <br>거래 정보, 주변 인프라, 환경 정보, 리뷰 한눈에 확인​<br><br>
**지역 통계 제공**
  <br>지역별 교통량, 범죄율, 소음 등​​<br><br>
**건물 찜하기, 사용자 건물 리뷰 기능**​<br><br>
**건물과 직장까지의 손쉬운 길 찾기 지원** (카카오맵, 네이버지도 앱 연결)
